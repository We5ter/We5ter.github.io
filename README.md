### [🔍 Try it now →](https://lightrains.org/android-osv/)

A **fully searchable** Android vulnerability database powered by **Google OSV**, containing **3,261+ entries** (ASB + PUB) with real-time filtering by severity, keyword, CVE ID, component name, and more.

#### ✨ Why this over the official [OSV.dev](https://osv.dev/)?

| Feature | **This Tool** ([android-osv](https://lightrains.org/android-osv/)) | Official [OSV.dev](https://osv.dev/) |
|---------|-------------------------------------------------------|----------------------------------|
| **Full-text search** | ✅ Search by **any keyword** (component, description, CVE, AOSP ID...) | ❌ Only vulnerability ID lookup |
| **Severity filter** | ✅ One-click filter: Critical / High / Moderate / Low | ❌ No severity-based filtering |
| **Offline-capable** | ✅ All data loaded client-side, instant response | ❌ Requires server round-trip per query |
| **Mobile-friendly** | ✅ Responsive design with hamburger nav | ⚠️ Limited on mobile |
| **Data coverage** | ✅ **3,261 entries** (ASB-A-* + legacy PUB-A-*) | Same source, different UX |

> 💡 Data sourced from [Google OSV](https://osv.dev/) · Auto-updated bi-monthly via GitHub Actions
